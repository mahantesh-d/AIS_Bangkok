Kubernetes health check
kubectl cluster-info

To view the nodes in the cluster using 
kubectl get nodes

To deploy--
kubectl run first-deploy --image=katacoda/docker-http-server --port=80

To Check the running pods
kubectl get pods

To expose the port
kubectl expose deployment first-deploy --port=80  --type=NodePort

The command below finds the allocated port and executes a HTTP request.
export PORT=$(kubectl get svc first-deploy -o go-template='{{range.spec.ports}}{{if .nodePort}}{{.nodePort}}{{"\n"}}{{end}}{{end}}')
curl host01:$PORT
